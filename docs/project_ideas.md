# Project Ideas

## 1. Makerspace Inventory Intelligence

Problem: small hardware rooms often manage parts across inconsistent spreadsheets, making it difficult to find assets, detect missing items, or decide what needs attention.

Why it is useful: it demonstrates data cleaning, search ranking, duplicate detection, visualization, CLI design, and testable Python code.

Possible extensions:

- interactive dashboard
- spreadsheet import templates
- barcode or QR-code label generation
- configurable category rules
- trend simulation for checkouts and returns

## 2. Hardware Room Audit Simulator

Problem: teams need a safe way to practice inventory audits without using sensitive operational records.

Why it is useful: it can generate synthetic asset lists, inject realistic inconsistencies, and evaluate how well audit rules catch missing or mislabeled items.

Possible extensions:

- synthetic data generator
- configurable error rates
- audit scoring report
- before/after cleanup comparison

## 3. Parts Search Benchmark

Problem: inventory search can be frustrating when item names are inconsistent or abbreviated.

Why it is useful: it compares search strategies such as exact matching, token overlap, trigram similarity, and weighted scoring.

Possible extensions:

- benchmark queries
- relevance labels
- precision and recall metrics
- small web demo
