# Data And Privacy Note

This repository uses synthetic and sanitized sample data for demonstration.

No personal records, confidential documents, access credentials, purchasing records, or unpublished operational exports are included.

The public repository is designed to demonstrate inventory analytics techniques without exposing sensitive context. Any non-public materials used during local experimentation should remain outside the repository.

Before publishing future changes, run a text scan for contact details, account credentials, names, internal locations, and large binary artifacts.
